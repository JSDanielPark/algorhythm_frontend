Development Process
-------------
We use Scrum(Agile) process

> **using:**

> - Sprint : Sprint cycle 1week
> - User Story : Who, What, Why
> - Task kanban : Todo, Doing, Do
> - Planing Porker : Estimate workload(Sprint Backlog). We decided score beetwen 1 and 10
